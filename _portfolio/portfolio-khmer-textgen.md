Built a full-stack web application that uses a custom-trained LSTM model to generate Khmer text. The system supports multiple writing styles (formal, royal, casual) and processes input through character-level tokenization across an 85-character Khmer vocabulary. Handled machine learning model development and data collection/preparation for training the neural network on Khmer language datasets.

## Tech Stack

`Python` `PyTorch` `LSTM` `Data Collection` `Data Preparation` `Character-level Tokenization`

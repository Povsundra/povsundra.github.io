## How to Add Certificate Images

### **Folder Structure**
Create these folders in your `images/` directory:

```
images/
├── certificates/
│   ├── volunteer-certificates/
│   │   ├── digital-literacy-trainer.png
│   │   ├── math-tutor.png
│   │   ├── volunteer-cert-4.png
│   │   └── volunteer-cert-5.png
│   ├── technical-certificates/
│   └── awards/
│       └── environment-award.png
```

### **Steps to Add Your Certificates**

1. **Create folders:**
   ```
   📁 images/certificates/volunteer-certificates/
   📁 images/certificates/awards/
   ```

2. **Save certificate images** (PNG or JPG, recommended size: 600x400px or larger)

3. **Reference in markdown:**
   ```markdown
   ![Certificate Name](/images/certificates/folder-name/filename.png)
   ```

### **Recommended Image Specifications**
- **Format:** PNG or JPG
- **Size:** 600x400px minimum (800x600px recommended)
- **Quality:** High resolution (300+ DPI for prints)
- **Naming:** Use descriptive names with hyphens (e.g., `digital-literacy-trainer.png`)
